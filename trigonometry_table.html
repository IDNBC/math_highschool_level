三角関数の値

30°: \\( \\cos(30^\circ) = \\frac{\\sqrt{3}}{2} \\), \\( \\sin(30^\circ) = \\frac{1}{2} \\), \\( \\tan(30^\circ) = \\frac{1}{\\sqrt{3}} \\), \\( \\cot(30^\circ) = \\sqrt{3} \\), \\( \\csc(30^\circ) = 2 \\), \\( \\sec(30^\circ) = \\frac{2}{\\sqrt{3}} \\)
45°: \\( \\cos(45^\circ) = \\frac{\\sqrt{2}}{2} \\), \\( \\sin(45^\circ) = \\frac{\\sqrt{2}}{2} \\), \\( \\tan(45^\circ) = 1 \\), \\( \\cot(45^\circ) = 1 \\), \\( \\csc(45^\circ) = \\sqrt{2} \\), \\( \\sec(45^\circ) = \\sqrt{2} \\)
60°: \\( \\cos(60^\circ) = \\frac{1}{2} \\), \\( \\sin(60^\circ) = \\frac{\\sqrt{3}}{2} \\), \\( \\tan(60^\circ) = \\sqrt{3} \\), \\( \\cot(60^\circ) = \\frac{1}{\\sqrt{3}} \\), \\( \\csc(60^\circ) = \\frac{2}{\\sqrt{3}} \\), \\( \\sec(60^\circ) = 2 \\)
90°: \\( \\cos(90^\circ) = 0 \\), \\( \\sin(90^\circ) = 1 \\), \\( \\tan(90^\circ) = \\infty \\), \\( \\cot(90^\circ) = 0 \\), \\( \\csc(90^\circ) = 1 \\), \\( \\sec(90^\circ) = \\infty \\)
