三角関数の値

次の表は、角度 \(x\) を \(0\) から \(2\pi\) まで \( \frac{\pi}{6} \) ごとに分割し、それぞれの三角関数の値を示します：

三角関数の値 (角度ごと)

角度 (\(x\)) \(\sin x\) \(\cos x\) \(\tan x\) \(\csc x\) \(\sec x\) \(\cot x\)
\(0\) \(\sin(0) = 0\) \(\cos(0) = 1\) \(\tan(0) = 0\) \(\csc(0) = \text{undefined}\) \(\sec(0) = 1\) \(\cot(0) = \text{undefined}\)
\(\frac{\pi}{6}\) \(\sin\left(\frac{\pi}{6}\right) = \frac{1}{2}\) \(\cos\left(\frac{\pi}{6}\right) = \frac{\sqrt{3}}{2}\) \(\tan\left(\frac{\pi}{6}\right) = \frac{1}{\sqrt{3}}\) \(\csc\left(\frac{\pi}{6}\right) = 2\) \(\sec\left(\frac{\pi}{6}\right) = \frac{2}{\sqrt{3}}\) \(\cot\left(\frac{\pi}{6}\right) = \sqrt{3}\)
\(\frac{\pi}{4}\) \(\sin\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}\) \(\cos\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}\) \(\tan\left(\frac{\pi}{4}\right) = 1\) \(\csc\left(\frac{\pi}{4}\right) = \sqrt{2}\) \(\sec\left(\frac{\pi}{4}\right) = \sqrt{2}\) \(\cot\left(\frac{\pi}{4}\right) = 1\)
\(\frac{\pi}{3}\) \(\sin\left(\frac{\pi}{3}\right) = \frac{\sqrt{3}}{2}\) \(\cos\left(\frac{\pi}{3}\right) = \frac{1}{2}\) \(\tan\left(\frac{\pi}{3}\right) = \sqrt{3}\) \(\csc\left(\frac{\pi}{3}\right) = \frac{2}{\sqrt{3}}\) \(\sec\left(\frac{\pi}{3}\right) = 2\) \(\cot\left(\frac{\pi}{3}\right) = \frac{1}{\sqrt{3}}\)
\(\frac{\pi}{2}\) \(\sin\left(\frac{\pi}{2}\right) = 1\) \(\cos\left(\frac{\pi}{2}\right) = 0\) \(\tan\left(\frac{\pi}{2}\right) = \text{undefined}\) \(\csc\left(\frac{\pi}{2}\right) = 1\) \(\sec\left(\frac{\pi}{2}\right) = \text{undefined}\) \(\cot\left(\frac{\pi}{2}\right) = 0\)
\(\frac{2\pi}{3}\) \(\sin\left(\frac{2\pi}{3}\right) = \frac{\sqrt{3}}{2}\) \(\cos\left(\frac{2\pi}{3}\right) = -\frac{1}{2}\) \(\tan\left(\frac{2\pi}{3}\right) = -\sqrt{3}\) \(\csc\left(\frac{2\pi}{3}\right) = \frac{2}{\sqrt{3}}\) \(\sec\left(\frac{2\pi}{3}\right) = -2\) \(\cot\left(\frac{2\pi}{3}\right) = -\frac{1}{\sqrt{3}}\)
\(\frac{3\pi}{4}\) \(\sin\left(\frac{3\pi}{4}\right) = \frac{\sqrt{2}}{2}\) \(\cos\left(\frac{3\pi}{4}\right) = -\frac{\sqrt{2}}{2}\) \(\tan\left(\frac{3\pi}{4}\right) = -1\) \(\csc\left(\frac{3\pi}{4}\right) = \sqrt{2}\) \(\sec\left(\frac{3\pi}{4}\right) = -\sqrt{2}\) \(\cot\left(\frac{3\pi}{4}\right) = -1\)
\(\frac{5\pi}{6}\) \(\sin\left(\frac{5\pi}{6}\right) = \frac{1}{2}\) \(\cos\left(\frac{5\pi}{6}\right) = -\frac{\sqrt{3}}{2}\) \(\tan\left(\frac{5\pi}{6}\right) = -\frac{1}{\sqrt{3}}\) \(\csc\left(\frac{5\pi}{6}\right) = 2\) \(\sec\left(\frac{5\pi}{6}\right) = -\frac{2}{\sqrt{3}}\) \(\cot\left(\frac{5\pi}{6}\right) = -\sqrt{3}\)
\(\pi\) \(\sin(\pi) = 0\) \(\cos(\pi) = -1\) \(\tan(\pi) = 0\) \(\csc(\pi) = \text{undefined}\) \(\sec(\pi) = -1\) \(\cot(\pi) = \text{undefined}\)
\(\frac{7\pi}{6}\) \(\sin\left(\frac{7\pi}{6}\right) = -\frac{1}{2}\) \(\cos\left(\frac{7\pi}{6}\right) = -\frac{\sqrt{3}}{2}\) \(\tan\left(\frac{7\pi}{6}\right) = \frac{1}{\sqrt{3}}\) \(\csc\left(\frac{7\pi}{6}\right) = -2\) \(\sec\left(\frac{7\pi}{6}\right) = -\frac{2}{\sqrt{3}}\) \(\cot\left(\frac{7\pi}{6}\right) = \sqrt{3}\)
\(\frac{5\pi}{4}\) \(\sin\left(\frac{5\pi}{4}\right) = -\frac{\sqrt{2}}{2}\) \(\cos\left(\frac{5\pi}{4}\right) = -\frac{\sqrt{2}}{2}\) \(\tan\left(\frac{5\pi}{4}\right) = 1\) \(\csc\left(\frac{5\pi}{4}\right) = -\sqrt{2}\) \(\sec\left(\frac{5\pi}{4}\right) = -\sqrt{2}\) \(\cot\left(\frac{5\pi}{4}\right) = 1\)
\(\frac{4\pi}{3}\) \(\sin\left(\frac{4\pi}{3}\right) = -\frac{\sqrt{3}}{2}\) \(\cos\left(\frac{4\pi}{3}\right) = -\frac{1}{2}\) \(\tan\left(\frac{4\pi}{3}\right) = \sqrt{3}\) \(\csc\left(\frac{4\pi}{3}\right) = -\frac{2}{\sqrt{3}}\) \(\sec\left(\frac{4\pi}{3}\right) = -2\) \(\cot\left(\frac{4\pi}{3}\right) = \frac{1}{\sqrt{3}}\)
\(\frac{3\pi}{2}\) \(\sin\left(\frac{3\pi}{2}\right) = -1\) \(\cos\left(\frac{3\pi}{2}\right) = 0\) \(\tan\left(\frac{3\pi}{2}\right) = \text{undefined}\) \(\csc\left(\frac{3\pi}{2}\right) = -1\) \(\sec\left(\frac{3\pi}{2}\right) = \text{undefined}\) \(\cot\left(\frac{3\pi}{2}\right) = 0\)
\(\frac{5\pi}{3}\) \(\sin\left(\frac{5\pi}{3}\right) = -\frac{\sqrt{3}}{2}\) \(\cos\left(\frac{5\pi}{3}\right) = \frac{1}{2}\) \(\tan\left(\frac{5\pi}{3}\right) = -\frac{1}{\sqrt{3}}\) \(\csc\left(\frac{5\pi}{3}\right) = -\frac{2}{\sqrt{3}}\) \(\sec\left(\frac{5\pi}{3}\right) = 2\) \(\cot\left(\frac{5\pi}{3}\right) = -\sqrt{3}\)
\(\frac{7\pi}{4}\) \(\sin\left(\frac{7\pi}{4}\right) = -\frac{\sqrt{2}}{2}\) \(\cos\left(\frac{7\pi}{4}\right) = \frac{\sqrt{2}}{2}\) \(\tan\left(\frac{7\pi}{4}\right) = -1\) \(\csc\left(\frac{7\pi}{4}\right) = -\sqrt{2}\) \(\sec\left(\frac{7\pi}{4}\right) = \sqrt{2}\) \(\cot\left(\frac{7\pi}{4}\right) = -1\)
\(\frac{11\pi}{6}\) \(\sin\left(\frac{11\pi}{6}\right) = -\frac{1}{2}\) \(\cos\left(\frac{11\pi}{6}\right) = \frac{\sqrt{3}}{2}\) \(\tan\left(\frac{11\pi}{6}\right) = -\frac{1}{\sqrt{3}}\) \(\csc\left(\frac{11\pi}{6}\right) = -2\) \(\sec\left(\frac{11\pi}{6}\right) = \frac{2}{\sqrt{3}}\) \(\cot\left(\frac{11\pi}{6}\right) = -\sqrt{3}\)
\(2\pi\) \(\sin(2\pi) = 0\) \(\cos(2\pi) = 1\) \(\tan(2\pi) = 0\) \(\csc(2\pi) = \text{undefined}\) \(\sec(2\pi) = 1\) \(\cot(2\pi) = \text{undefined}\)
