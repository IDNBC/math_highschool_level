次の問いに答えよ。

1. \( a, b \in \mathbb{R} \, (a < b) \) に対して

\[ f'(a)(b-a) \leq f(b) - f(a) \leq f'(b)(b-a) \] を示せ。

2. \( a, b \in \mathbb{R} \, (a < b) \) を固定する。

関数 \( F_{a,b} : [0, 1] \to \mathbb{R} \) を \[ F_{a,b}(t) = (1-t)f(a) + tf(b) - f((1-t)a + tb) \] と定めるとき \[ F_{a,b}(t) \geq 0 \quad (t \in [0, 1]) \] を示せ。

解答

  1. 平均値の定理から \( f(b) - f(a) = f'(c)(b-a) \) となる \( c \, (a < c < b) \) がある。
    従って、 \[ f'(a)(b-a) \leq f'(c)(b-a) = f(b) - f(a) \leq f'(b)(b-a) \]
  2. \[ F_{a,b}'(t) = f(b) - f(a) - f'((1-t)a + tb)(b-a), \quad F_{a,b}''(t) = -f''((1-t)a + tb)(b-a)^2 \] である。\( f' \) が単調増加なので \[ F_{a,b}''(t) = -f''((1-t)a + tb)(b-a)^2 \leq 0 \] である。つまり、\( F_{a,b}' \) は単調減少である。
    (1) より \( F_{a,b}(0) \geq 0 \), \( F_{a,b}(1) \leq 0 \), \( F_{a,b}(1) = F_{a,b}(0) = 0 \) である。
    以上のことを踏まえて、増減を調べることにより \( F_{a,b}(t) \geq 0 \, (t \in [0, 1]) \) となることがわかる。
