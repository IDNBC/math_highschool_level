ご利用方法

1. 数学コンテンツの著作権

当サイトに掲載されているすべてのコンテンツは、著作権法によって保護されています。著作権者の許可なく、当サイトの内容を複製、配布、または商用利用することはできません。教材や問題の一部を利用する場合は、必ず出典を明記してください。

2. 学習に必要な語学力

日本語力: 当サイトは主に日本語で提供されていますので、漢字の読解など基礎的な読解力があると効果的に学べます。

英語力: 高度な数学用語や海外の研究成果を理解する際に、ある程度の英語力が役立つ場合がありますが、必要に応じて補助教材を用意しています。

3. 学習者向けの数学レベル

初級者向け: 基礎的な算数や中学校レベルの数学から始められます。基礎的なことを改めて確認する意味合いの説明がほとんどなので、小学生向けの解説の仕方ではありません。

上級者向け: 高校・大学レベルの数学や、応用数学、線形代数、微分積分まで幅広く学べます。レベルに応じた問題を選べるようになっていますので、無理なくステップアップが可能です。

4. MathJaxによる数式表示

当サイトでは、複雑な数学的表現を視覚的にわかりやすくするために MathJax を使用しています。スムーズな数式表示を行うために、以下の点にご注意ください。

  • JavaScriptを有効にする: MathJax は JavaScript によって動作します。お使いのブラウザで JavaScript の設定をオンにしてください。
  • 互換性のあるブラウザ: MathJax はほとんどの現代的なウェブブラウザに対応していますが、できるだけ最新バージョンをご使用ください。

MathJax の動作確認

ブラウザ設定が正しく行われている場合、以下のような数式がきれいに表示されます：

$$E = mc^2$$

表示に問題がある場合は、ブラウザの設定を見直してください。
