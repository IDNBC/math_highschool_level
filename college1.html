実数の公理

数学における公理（こうり、axiom）とは、証明を必要とせずに真とされる基本的な命題や原理のことです。公理は理論の基礎を成し、それに基づいて他の全ての命題が証明されます。実数の公理も、その一例です。

実数の公理のほかにも次のようなものがあります：

  • ユークリッド幾何学の公理：例えば、二点を結ぶ直線が唯一存在すること。
  • 集合論の公理：例えば、任意の集合に対して、その部分集合の集合が存在すること。
  • 代数学の公理：例えば、ゼロ元や単位元の存在。

実数の公理

実数（じっすう、real number）は、整数、有理数、および無理数を含む数の集合です。実数には次のような基本的な性質（公理）があります：

  1. 加法の公理：任意の実数 \(a\) と \(b\) に対して、\(a + b\) は実数である。
  2. 乗法の公理：任意の実数 \(a\) と \(b\) に対して、\(a \times b\) は実数である。
  3. 分配法則：任意の実数 \(a\)、\(b\)、および \(c\) に対して、\(a \times (b + c) = (a \times b) + (a \times c)\)。
  4. 加法の単位元の存在：任意の実数 \(a\) に対して、\(a + 0 = a\)。
  5. 乗法の単位元の存在：任意の実数 \(a\) に対して、\(a \times 1 = a\)。
  6. 加法の逆元の存在：任意の実数 \(a\) に対して、ある実数 \(-a\) が存在し、\(a + (-a) = 0\)。
  7. 乗法の逆元の存在：0でない任意の実数 \(a\) に対して、ある実数 \(a^{-1}\) が存在し、\(a \times a^{-1} = 1\)。

例題

次の実数の性質を確認してみましょう：

  1. 加法の単位元：\(5 + 0\)
  2. 乗法の単位元：\(3 \times 1\)
  3. 加法の逆元：\(7 + (-7)\)
  4. 分配法則：\(2 \times (3 + 4)\)

解答

  1. \(5 + 0 = 5\)（加法の単位元）
  2. \(3 \times 1 = 3\)（乗法の単位元）
  3. \(7 + (-7) = 0\)（加法の逆元）
  4. \(2 \times (3 + 4) = 2 \times 7 = 14\)（分配法則）

練習問題

次の実数の性質を確認してください：

  1. 加法の単位元：\(12 + 0\)
  2. 乗法の単位元：\(6 \times 1\)
  3. 加法の逆元：\(9 + (-9)\)
  4. 分配法則：\(5 \times (2 + 3)\)
  5. 乗法の逆元：\(4 \times 4^{-1}\)
解答を表示/非表示
  1. \(12 + 0 = 12\)（加法の単位元）
  2. \(6 \times 1 = 6\)（乗法の単位元）
  3. \(9 + (-9) = 0\)（加法の逆元）
  4. \(5 \times (2 + 3) = 5 \times 5 = 25\)（分配法則）
  5. \(4 \times 4^{-1} = 4 \times \frac{1}{4} = 1\)（乗法の逆元）
