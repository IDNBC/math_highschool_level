分数と逆数の計算の法則

1. 分数の加法と減法

分母が同じ分数同士は、分子を加減する:

  • 分母が同じ場合: 分子同士を加減 (例: \(\frac{3}{4} + \frac{2}{4} = \frac{5}{4}\))
  • 分母が異なる場合: 公倍数を使って通分する (例: \(\frac{1}{3} + \frac{1}{6} = \frac{2}{6} + \frac{1}{6} = \frac{3}{6} = \frac{1}{2}\))

例題

\(\frac{5}{8} + \frac{3}{8} = \frac{8}{8} = 1\)

\(\frac{2}{5} - \frac{1}{5} = \frac{1}{5}\)

\(\frac{1}{4} + \frac{1}{2} = \frac{1}{4} + \frac{2}{4} = \frac{3}{4}\)

2. 分数の乗法

分数同士の掛け算は、分子同士、分母同士を掛ける:

  • \(\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}\) (例: \(\frac{2}{3} \times \frac{4}{5} = \frac{2 \times 4}{3 \times 5} = \frac{8}{15}\))

例題

\(\frac{3}{7} \times \frac{2}{5} = \frac{6}{35}\)

\(\frac{4}{9} \times \frac{3}{8} = \frac{12}{72} = \frac{1}{6}\)

\(\frac{5}{6} \times \frac{6}{5} = 1\)

3. 分数の除法

分数同士の割り算は、割る数の逆数を掛ける:

  • \(\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}\) (例: \(\frac{3}{4} \div \frac{2}{5} = \frac{3}{4} \times \frac{5}{2} = \frac{15}{8}\))

例題

\(\frac{7}{9} \div \frac{2}{3} = \frac{7}{9} \times \frac{3}{2} = \frac{21}{18} = \frac{7}{6}\)

\(\frac{5}{8} \div \frac{1}{4} = \frac{5}{8} \times 4 = \frac{20}{8} = \frac{5}{2}\)

\(\frac{9}{10} \div \frac{3}{5} = \frac{9}{10} \times \frac{5}{3} = \frac{45}{30} = \frac{3}{2}\)

4. 逆数の計算

ある数 \(a\) の逆数は \(\frac{1}{a}\) であり、逆数同士を掛けると 1 になる:

  • 例: \(5\) の逆数は \(\frac{1}{5}\)、 \(\frac{3}{4}\) の逆数は \(\frac{4}{3}\)
  • \(a \times \frac{1}{a} = 1\) (例: \(7 \times \frac{1}{7} = 1\))

例題

\(4\) の逆数は \(\frac{1}{4}\)

\(\frac{2}{5}\) の逆数は \(\frac{5}{2}\)

\(8 \times \frac{1}{8} = 1\)

練習問題

問題 1: 分数の加法と減法

a. \(\frac{2}{3} + \frac{1}{6} = ?\)

b. \(\frac{5}{8} - \frac{1}{4} = ?\)

c. \(\frac{3}{5} + \frac{2}{5} = ?\)

解答を表示/非表示

解答

a. \(\frac{2}{3} + \frac{1}{6} = \frac{4}{6} + \frac{1}{6} = \frac{5}{6}\)

b. \(\frac{5}{8} - \frac{1}{4} = \frac{5}{8} - \frac{2}{8} = \frac{3}{8}\)

c. \(\frac{3}{5} + \frac{2}{5} = \frac{5}{5} = 1\)

問題 2: 分数の乗法

a. \(\frac{3}{7} \times \frac{2}{5} = ?\)

b. \(\frac{4}{9} \times \frac{3}{8} = ?\)

c. \(\frac{5}{6} \times \frac{6}{5} = ?\)

解答を表示/非表示

解答

a. \(\frac{3}{7} \times \frac{2}{5} = \frac{6}{35}\)

b. \(\frac{4}{9} \times \frac{3}{8} = \frac{12}{72} = \frac{1}{6}\)

c. \(\frac{5}{6} \times \frac{6}{5} = 1\)

問題 3: 分数の除法

a. \(\frac{8}{9} \div \frac{2}{3} = ?\)

b. \(\frac{7}{10} \div \frac{1}{2} = ?\)

c. \(\frac{4}{5} \div \frac{2}{3} = ?\)

解答を表示/非表示

解答

a. \(\frac{8}{9} \div \frac{2}{3} = \frac{8}{9} \times \frac{3}{2} = \frac{24}{18} = \frac{4}{3}\)

b. \(\frac{7}{10} \div \frac{1}{2} = \frac{7}{10} \times 2 = \frac{14}{10} = \frac{7}{5}\)

c. \(\frac{4}{5} \div \frac{2}{3} = \frac{4}{5} \times \frac{3}{2} = \frac{12}{10} = \frac{6}{5}\)

c. \(6 \times \frac{1}{6} = ?\)

c. \(6 \times \frac{1}{6} = 1\)
