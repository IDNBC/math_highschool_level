2重根号の外し方

2重根号とは、根号の中にさらに根号が入っている形式のことで、以下のように表されます：

\[ \sqrt{a + \sqrt{b}} \]

2重根号を外す方法は、以下の形式に変換することにあります。

\[ \sqrt{a + \sqrt{b}} = \sqrt{x} + \sqrt{y} \]

これを成立させるためには、以下の条件を満たす \(x\) と \(y\) を見つける必要があります。

\[ x + y = a \quad \text{かつ} \quad 2\sqrt{xy} = \sqrt{b} \]

方法

  1. \(a + \sqrt{b} = x + y + 2\sqrt{xy}\)
  2. それぞれの条件 \(x + y = a\) と \(2\sqrt{xy} = \sqrt{b}\) を解く

簡単な例題

次の2重根号を外してみましょう： \[ \sqrt{3 + \sqrt{5}} \]

  1. まず、\(\sqrt{3 + \sqrt{5}} = \sqrt{x} + \sqrt{y}\) と仮定します。
  2. 次に、\(x + y = 3\) かつ \(2\sqrt{xy} = \sqrt{5}\) という式を立てます。
  3. ここで、\(x\) と \(y\) を見つけるためにそれぞれの式を解きます。例えば、\(x\) と \(y\) に適した値を見つけることで、 \[ 2\sqrt{xy} = \sqrt{5} \implies xy = \frac{5}{4} \] \[ x + y = 3 \]
  4. 解の公式を使って方程式を解くと、 \[ x, y = \frac{3 \pm 1}{2} \implies x = 2, y = 1 \quad \text{または} \quad x = 1, y = 2 \]
  5. これより、 \[ \sqrt{3 + \sqrt{5}} = \sqrt{2} + \sqrt{1} \]
  6. 結果として、\(\sqrt{3 + \sqrt{5}} = \sqrt{2} + 1\) となります。

練習問題

以下の2重根号を外してください。

  1. \(\sqrt{6 + \sqrt{20}}\)
  2. \(\sqrt{10 + \sqrt{24}}\)
  3. \(\sqrt{8 + \sqrt{15}}\)
  4. \(\sqrt{12 + \sqrt{40}}\)
  5. \(\sqrt{5 + \sqrt{21}}\)
解答を表示/非表示
  1. \[ \sqrt{6 + \sqrt{20}} = \sqrt{5} + \sqrt{1} \] (条件: \(x + y = 6\) と \(2\sqrt{xy} = \sqrt{20}\), \(xy = 5\))
  2. \[ \sqrt{10 + \sqrt{24}} = \sqrt{6} + \sqrt{4} \] (条件: \(x + y = 10\) と \(2\sqrt{xy} = \sqrt{24}\), \(xy = 6\))
  3. \[ \sqrt{8 + \sqrt{15}} = \sqrt{5} + \sqrt{3} \] (条件: \(x + y = 8\) と \(2\sqrt{xy} = \sqrt{15}\), \(xy = \frac{15}{4} \))
  4. \[ \sqrt{12 + \sqrt{40}} = \sqrt{8} + \sqrt{4} \] (条件: \(x + y = 12\) と \(2\sqrt{xy} = \sqrt{40}\), \(xy = 8\))
  5. \[ \sqrt{5 + \sqrt{21}} = \sqrt{7} + \sqrt{-2} \quad \text{または} \quad \sqrt{7 - 2\sqrt{3}} \] (条件: \(x + y = 5\) と \(2\sqrt{xy} = \sqrt{21}\), \(xy = \frac{21}{4}\))
