実数の公理

実数の公理は数学の基礎を形成します。ここでは、順序の公理、完備性の公理、および算術の公理について説明します。また、例題と練習問題も含めています。

順序の公理

  • 反射律: 任意の実数 a について a ≤ a が成り立ちます。
  • 反対称律: 任意の実数 a, b について a ≤ b かつ b ≤ a ならば a = b が成り立ちます。
  • 推移律: 任意の実数 a, b, c について a ≤ b かつ b ≤ c ならば a ≤ c が成り立ちます。
  • 全順序律: 任意の実数 a, b について a ≤ b あるいは b ≤ a が成り立ちます。

完備性の公理

実数は 完備 です。すなわち、任意の有界非空集合は上限（最小上界）と下限（最大下界）を持ちます。

算術の公理

  • 閉性: 任意の実数 a, b について a + b と ab も実数です。
  • 交換律: 任意の実数 a, b について a + b = b + a および ab = ba が成り立ちます。
  • 結合律: 任意の実数 a, b, c について (a + b) + c = a + (b + c) および (ab)c = a(bc) が成り立ちます。
  • 分配律: 任意の実数 a, b, c について a(b + c) = ab + ac が成り立ちます。
  • 単位元の存在: 加法の単位元は 0、乗法の単位元は 1 です。
  • 逆元の存在: 任意の実数 a について加法の逆元は -a、乗法の逆元は 1/a（a ≠ 0 の場合）です。

例題

例題 1: a = 3、b = 4 のとき、a + b と ab を求めなさい。

解答:

  • a + b = 3 + 4 = 7
  • ab = 3 × 4 = 12

練習問題

  1. a = 2、b = 5 のとき、a + b と ab を求めなさい。
  2. 実数集合 {1, 2, 3, 4, 5} の上限と下限を求めなさい。
  3. a = -3、b = 7 のとき、a と b の和と積を求めなさい。

解答

  1. a + b = 2 + 5 = 7、ab = 2 × 5 = 10
  2. 上限は 5、下限は 1 です。
  3. a + b = -3 + 7 = 4、ab = -3 × 7 = -21
